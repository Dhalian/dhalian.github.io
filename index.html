What is Homebrew ?

Homebrew applications are unlicensed software made for closed systems such as the PS2. These applications can range from utilities to custom homebrew games

What is FreeMCBoot ?

FreeMCBoot is an open source Homebrew loader-Replacement for the Home Menu of your PS2, you launch it from a MagicGate Memory Card, it can be installed with other methods such as FreeDVDBoot, FreeHDBoot

What can i do by modding my system ?

  • Launch ISOs from your USB Drive using OPL(OpenPS2Loader)
  • Use emulators such as RetroArch
  • Play Homebrew games

What this guide will do to my system ?

This guide will not really change your system, it will install FreeMCBoot to your Memory Card, nothing is installed to your NAND

Is there any brick risk ?

Actually, no, there is nothing installed on the NAND, so there isn't any risk of any brick or system damages

Which method should i use ?

  • If you have a Memory Card that can launch FreeMCBoot (generally there's a label with the FMCB v.1.9xx mention and a USB Drive
    • You need to use the basic FreeMCBoot method
  • If you have a DVD-Rom Disk, a PS2 with a compatible DVD Reader, a x32 laptop and a USB Drive
    • You need to use the FreeDVDBoot method
  • NOTE : to check the DVD Reader version of your PS2 : on the system menu : click on .
  • To check if your DVD Reader is supported : check the FreeDVD Boot Compatibility List
  • If you haven't any of these accessories, it's recommended to buy a FMCB Memory Card and use the FreeMCBoot method which is the easier method and the cheapest.
